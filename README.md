# Personal Finance Tracker

## Overview

The **Personal Finance Tracker** is a Python command-line application that helps users record, manage, and analyze their expenses.
Users can add expenses, view them, generate reports, and store the data in a JSON file.

This project demonstrates Python concepts such as:

* Classes and objects
* File handling (JSON)
* Modular programming
* Testing with pytest

---

## Features

* Add a new expense (date, amount, category, description)
* View all saved expenses
* Generate a summary report
* Calculate total monthly expenses
* Backup expense data
* Save and load data using JSON
